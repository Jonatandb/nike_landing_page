## Run Locally

Clone the project

```bash
  git clone https://github.com/jonatandb/nike_landing_page/
```

Go to the project directory

```bash
  cd nike_landing_page
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```



## Tech Stack

**Client:** React, Vite, TailwindCSS
